# Fundamentos Java

Este repositório contém exercícios e códigos de prática dos fundamentos da linguagem Java.

## 📁 Estrutura
- `src/exercicios`: contém os exercícios organizados por tópicos básicos, como variáveis, condicionais, laços, métodos, arrays, etc.

## 🛠️ Tecnologias utilizadas
- Java JDK 17+
- IntelliJ IDEA

## 🎯 Objetivo
Praticar e reforçar os conceitos principais da programação em Java, com foco na lógica de programação e estrutura de código limpa.
